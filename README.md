# Nick

A BeamMP mod that lets players set a custom nickname for their vehicles, visible to other players.

Made by [Neptnium](https://github.com/Neptnium) — Discord: `@neptnium`

## Usage

In the in-game chat, type:

```
/nick <your nickname>
```

Your nickname will be displayed above all the vehicles you own. New players joining the server will automatically see your current nickname.

## Installation

### Server

Copy the `server/` folder contents into your BeamMP server's `Resources/Server/Nick/` directory.

### Client

Install `client.zip` as a BeamMP client mod (drop it in your BeamMP mods folder).
